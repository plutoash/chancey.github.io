《通信软件开发与应用》课程结业报告

开发内容


采用纯 CSS 语言等构建一个关于中华传统美食的网站（静态页面）
页面内容包括：首页，稀粥，饺子，饮料和大饼六个页面和子页面
通过点击美食可以查看具体的制作步骤

开发过程


采用纯 CSS 语言
插入大量图片与css结合使网页更美观，部分图片处理和图层使用ps进行制作
使用浮动图标和超链接实现页面之间 的跳转

遇到的问题


1.图片插入到位置和大小如何设置（CSS布局中对于图片列表或图片排版时，图片不是固定宽度高度大小，但图片占位是固定宽度高度
这个时候如果使用CSS固定死图片大小，这个时候如果图片相不是等比例大小，那么这张图片就会变形不清晰）
2.怎样布局使文字 与图片完美融合
3.如何制作流畅的滑动等动画效果
etc

解决过程及疑问


1.最好解决方法就是从图片本身出发，将图片设计成与布局中宽度高度成等比例图片，才能保证真正图片缩小后不变形，显示完整。 或者使用代码让图片和布局高度宽度等比例缩放
2.常见方法有3种：一.通过添加css的“vertical-align:middle;”；二.如果图片是背景图片，可以在css中设置背景图片；
三.把文字和图片分别放入不同的div中。 上面三种方法都可以让图片和文字在同一行显示
3.引入关键属性 : -webkit-overflow-scrolling:touch;
